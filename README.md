# Sage - Tailwind Boilerplate

- Sage 9.0.9
- Tailwind 1.5.2

## Config

Edit Sage config in `resources/assets/config.json`

```
...
"publicPath": "/wp-content/themes/sage",
"devUrl": "http://wordpress.test",
...

```

## Install

    yarn

## Dev Server

    yarn start

## Build

    yarn build
    
## Theme name

Edit theme name in `resources/style.css`
